CPU

컴퓨터의 두뇌 역할
성능을 좌우하는 가장 중요한 요소

CPU 성능을 결정하는 3가지

코어

코어는 cpu에 내장된 핵심 부품으로, 예전에는 1개의 CPU 당 1 개의 코어만을 가지고 있었다. 하지만 2005년 듀올 코어의 등장으로 다중 코어 cpu의 대중화가 시작되었다. 최근에는 6코어, 8코어, 16코어 등 하나의 cpu에 여러 개의 코어가 포함되어 있다. 다중 코어 cpu는 내장된 코어의 수와 같은 수의 단일 코어 cpu를 동시에 설치한 것과 유사한 성능을 낼 수 있다.

클럭

CPU의 연산 속도는 CPU의 종류마다 다르다. 속도를 나타내는 대표적인 단위가 '클럭'인데 이는 1초당 cpu 내부에서 몇 단계의 작업이 처리되는 지를 측정해 이를 주파수 단위인 'HZ(헤르츠)'로 나타낸 것이다. 클럭 수치가 높을수록 빠른 성능의 CPU라고 할 수 있다. 단일 코어인 경우 클럭 속도가 CPU의 성능을 나타내는 절대적인 기준이었지만, 멀티 코어의 경우에는 코어의 수와 더불어 성능을 나타내는 기준이 되고 있다.

캐시메모리

CPU의 등급과 가격을 구분 짓는 중요한 지표로 쓰이는 것이 캐시 메모리이다. 줄여서 '캐시'라 부르는데 CPU가 데이터를 처리할 때 자주 사용하는 데이터를 임시 보관하는 곳이다. 캐시 메모리의 용량이 작으면 CPU에 비해 동작 속도가 훨씬 느린 기억장치에서 직접 데이터를 불러들어야 하는데 이는 컴퓨터의 전반적인 처리 속도를 크게 저하시킨다. 따라서 캐시 메모리가 클수록 성능 향상에 유리하다.
